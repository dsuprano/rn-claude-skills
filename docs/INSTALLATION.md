# Installation

## Requisitos
- Git
- Claude Code instalado

## Instalación global
Instala las skills para todos tus proyectos:

```bash
./install-global.sh
```

Destino:
```bash
~/.claude/skills/
```

## Instalación por proyecto
Instala las skills solo en el repo actual:

```bash
./install-project.sh
```

Destino:
```bash
.claude/skills/
```

## Verificación
Comprueba que existen carpetas de skill con `SKILL.md` dentro.
