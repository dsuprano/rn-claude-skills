# Contributing

## Cómo crear una nueva skill
1. Ejecutar:
   ```bash
   ./new-skill.sh nombre-skill
   ```
2. Editar `SKILL.md`
3. Añadir docs, ejemplos y scripts si hacen falta
4. Ejecutar:
   ```bash
   ./validate.sh
   ```
5. Hacer commit

## Reglas
- no meter todo en `SKILL.md`
- mantener description clara
- usar nombres consistentes
- añadir ejemplos si la skill es ambigua
